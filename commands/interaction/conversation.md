> [!NOTE]
> 以下指令仅群聊场景可用

## 香草迪拉熊

会被迪拉熊骂

```sh
(香草|xc)(迪拉熊|dlx)
```

#### 示例

```sh
xcdlx
```

## 喜欢你

将获得迪拉熊的喜爱

```sh
(迪拉熊|dlx)(我喜欢你|wxhn)
```

#### 示例

```sh
dlxwxhn
```

## 随机选择

迪拉熊帮你选

```sh
(迪拉熊|dlx)...是<option:string>(还是<option:string>)+
```

#### 示例

```sh
迪拉熊你是熊还是狮子还是沃尔玛购物袋
```

## 智能对话

触发智能对话功能

> [!NOTE]
> 私聊场景无需@迪拉熊即可使用

> [!WARNING]
> 使用普通版其他指令请勿@迪拉熊

```sh
@迪拉熊 <message:string>
```

#### 示例

```sh
@迪拉熊 你好
```

#### 参数

> `message`
>
> 消息
>
> ##### 类型
>
> 文字
>
> ##### 重要度
>
> 必填
>
> ##### 示例
>
> `你好`

## 复读

```sh
{三人连续重复同一消息}
```

## 文字转语音

将文本转换为迪拉熊语音

```sh
(迪拉熊|dlx)(说：|说|say|speak|tts|t2s|tta|t2a) <content:string>
```

#### 示例

```sh
dlxsay 你好，我是迪拉熊
```

#### 参数

> `content`
>
> 要转换的文本
>
> ##### 类型
>
> 文字
>
> ##### 重要度
>
> 必填
>
> ##### 示例
>
> `你好，我是迪拉熊`
