# 图库

> [!NOTE]
> 以下指令仅普通版可用

> [!NOTE]
> 以下指令仅群聊场景可用

## 随机作品

随机发送迪拉熊相关作品

# [普通图片](#tab/sfw)

```sh
(随机)?(迪拉熊|dlx)
```

#### 示例

```sh
dlx
```

# [NSFW图片](#tab/nsfw)

> [!NOTE]
> 该指令仅色色版可用

```sh
(随机)?(迪拉熊|dlx)(色图|st|nsfw)
```

#### 示例

```sh
dlxst
```

# [视频](#tab/video)

```sh
(随机)?(迪拉熊|dlx)(视频|sp|v)
```

#### 示例

```sh
dlxsp
```

## 厨力排行榜

获取迪拉熊图库查询次数排行榜

```sh
(迪拉熊|dlx)(厨力(排行榜|榜)|rank)
```

#### 示例

```sh
dlxrank
```
