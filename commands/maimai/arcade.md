# 机厅排卡/人数查询

> [!NOTE]
> 以下指令仅群聊场景可用

## 注册机厅

注册并为当前聊天绑定机厅

# [官 B 版](#tab/qq)

```sh
@迪拉熊 /管理机厅 注册 <name:string>
```

#### 示例

```sh
@迪拉熊 /管理机厅 注册 xxx机厅（xxx店）
```

# [普通版](#tab/ob11)

```sh
注册机厅 <name:string>
```

#### 示例

```sh
注册机厅 xxx机厅（xxx店）
```

---

#### 参数

> `name`
>
> 机厅全称
>
> ##### 类型
>
> 文字
>
> ##### 重要度
>
> 必填
>
> ##### 示例
>
> `xxx机厅（xxx店）`

```

```
