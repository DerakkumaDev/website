# 评分对象曲目（B50）

> [!TIP]
> 普通版以下指令支持在命令中附带`@某人`来代替他人查询

## 标准

生成标准评分对象曲目列表（B50）

> [!TIP]
> 这里的标准并非指代标准乐曲

# [官B版](#tab/qq)

```sh
@迪拉熊 /(评分对象曲目|b50) 标准
```

#### 示例

```sh
@迪拉熊 /b50 标准
```

# [普通版](#tab/ob11)

```sh
(迪拉熊|dlx)(b50|50)
```

#### 示例

```sh
dlx50
```

> `value`
>
> DX星星数量

>

>
> 必填
>
> ##### 示例
>
> `3 4 5`

## 偷师

生成由双方都存在成绩的谱面组成的评分对象曲目列表

> [!TIP]
> 排序规则为：
>
> 先选出“被查询者”分数比“查询者”最低B35/B15分还高的曲目，然后按照“被查询者分数”减去“查询者分数”的差值，从大到小排列。

> [!NOTE]
> 该指令仅普通版可用

> [!NOTE]
> 该指令要求在命令中附带`@某人`

```sh
(迪拉熊|dlx)cf(50)?
```

#### 示例

```sh
dlxcf
```

>
> `15000`

## 按寸止筛选

生成使用寸止进行筛选的评分对象曲目列表

> [!TIP]
> 寸止：成绩刚好卡在下个评级线附近，差一点点就能升段，范围是“下个评级的最低达成率”减去（“绝赞50落的101-达成率”和“TAP粉的101-达成率”里较小的那个的绝对值）到“下个评级的最低达成率”之间。

# [官B版](#tab/qq)

```sh
@迪拉熊 /(评分对象曲目|b50) 寸止
```

#### 示例

```sh
@迪拉熊 /b50 寸止
```

# [普通版](#tab/ob11)

```sh
(迪拉熊|dlx)(sunn|sun|cun|寸|🤏)(50)?
```

#### 示例

```sh
dlxcun
```

## 按锁血筛选

> [!TIP]
> 锁血：成绩刚好卡在“当前评级的最低达成率”到“当前评级的最低达成率 + 一小段范围”之间，这段范围取“绝赞50落的101-达成率”和“TAP粉的101-达成率”里较小的那个的绝对值。

# [官B版](#tab/qq)
