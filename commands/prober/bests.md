# 评分对象曲目（B50）

> [!TIP]
> 普通版以下指令支持在命令中附带`@某人`来代替他人查询

## 标准

生成标准评分对象曲目列表（B50）

> [!TIP]
> 这里的标准并非指代标准乐曲

# [官 B 版](#tab/qq)

```sh
@迪拉熊 /(评分对象曲目|b50) 标准
```

#### 示例

```sh
@迪拉熊 /b50 标准
```

# [普通版](#tab/ob11)

```sh
(迪拉熊|dlx)(b50|50)
```

#### 示例

```sh
dlx50
```

---

## 按 DX 星数筛选


# [普通版](#tab/ob11)

```sh

```

#### 示例

```sh

```

---

#### 参数

> `value`
>
> DX 星星数量
>
> ##### 类型
>
> 单或多个整数（0~5）
>
> ##### 重要度
>
> 必填
>
> ##### 示例
>
> `3 4 5`

## 全理论

> 该指令仅支持水鱼源

# [官 B 版](#tab/qq)

```sh
@迪拉熊 /(评分对象曲目|b50) 理论 [rating:number]
```

#### 示例

```sh
@迪拉熊 /b50 理论 15000
```

# [普通版](#tab/ob11)

```sh
(迪拉熊|dlx)(max|rr|ll|ma)(50)? [rating:number]
```

#### 示例

```sh
dlxll 15000
```

---

#### 参数

> `rating`
>
> 最高 DX 评分，如果无法达成则向下约分
>
> ##### 类型
>
> 整数（大于 0）
>
> ##### 重要度
>
> 可选
>
> ##### 示例
>
> `15000`
