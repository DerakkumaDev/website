# 开字母

> [!NOTE]
> 以下指令仅普通版可用

> [!NOTE]
> 以下指令仅群聊场景可用

## 开始游戏

开始开字母游戏

```sh
(开始|ks)(猜歌|开字母|kzm)
```

#### 示例

```sh
kskzm
```

## 开出字母

开出可能存在于歌名中的字母

```sh
开<word:string>
```

#### 示例

```sh
开a
```

## 尝试猜歌

尝试直接猜出乐曲

```sh
<name:string>
```

## 结束游戏

结束当前开字母游戏

```sh
(结束|跳过|js|tg)(猜歌|开字母|kzm)
```

#### 示例

```sh
jskzm
```

## 提示

获取乐曲相关的提示

# [乐曲信息](#tab/info)

> [!TIP]
> 每首乐曲有5次获取机会

```sh
(提示|提醒|信息) [index:number]
```

#### 示例

```sh
提示
```

# [封面](#tab/jacket)

> [!TIP]
> 每首乐曲有2次获取机会

```sh
(封面|曲绘|图片|图) [index:number]
```

#### 示例

```sh
封面
```

# [乐曲片段](#tab/slice)

> [!TIP]
> 每首乐曲有1次获取机会

```sh
(音乐|音频|乐曲|歌曲|片段) [index:number]
```

#### 示例

```sh
音乐
```

---

#### 参数

> `index`
>
> 序号
>
> ##### 类型
>
> 整数（1~5）
>
> ##### 重要度
>
> 可选
>
> ##### 示例
>
> `1`

## 排行榜

查询猜歌前十排名情况

```sh
(猜歌|开字母|kzm)(排行榜|榜|rank)
```

#### 示例

```sh
kzmrank
```

## 排名

查询猜歌个人排名情况

```sh
(猜歌|开字母|kzm)(个人排名|排名|th)
```

#### 示例

```sh
kzmth
```
