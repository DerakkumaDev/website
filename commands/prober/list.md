# 成绩列表

> [!TIP]
> 以下指令支持在命令后附带页码来翻页

## 按等级筛选

生成使用等级进行筛选的成绩列表

# [官 B 版](#tab/qq)

```sh
@迪拉熊 /成绩表 等级 <level:number>+?
```

#### 示例

```sh
@迪拉熊 /成绩表 等级 14+
```

# [普通版](#tab/ob11)

```sh
(list|comp|完成表|完成列表|完成表格) <level:number>+?
```

#### 示例

```sh
comp 14+
```

---

#### 参数

> `level`
>
> 等级
>
> ##### 类型
>
> 整数（1~15）
>
> ##### 重要度
>
> 必填
>
> ##### 示例
>
> `14`

## 按版本筛选

生成使用版本进行筛选的成绩列表

> [!NOTE]
> 该指令仅支持水鱼源

# [官 B 版](#tab/qq)

```sh
@迪拉熊 /成绩表 版本 (真|超|檄|橙|晓|桃|樱|紫|堇|白|雪|辉|舞|熊|华|爽|煌|宙|星|祭|祝|双|宴|镜|彩) (绿|黄|红|紫|白)?
```

#### 示例

```sh
@迪拉熊 /成绩表 版本 熊
```

# [普通版](#tab/ob11)

```sh
(list|comp|完成表|完成列表|完成表格) (真|超|檄|橙|晓|桃|樱|紫|堇|白|雪|辉|舞|熊|华|爽|煌|宙|星|祭|祝|双|宴|镜|彩) (绿|黄|红|紫|白)?
```

#### 示例

```sh

```
